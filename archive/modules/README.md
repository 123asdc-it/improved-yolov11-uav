# archive/modules/ — 归档模块目录

以下模块不再被活跃脚本 import，保留作为实现参考和改进尝试历史。

---

## `bifpn.py`

- **来源**：EfficientDet（CVPR 2020）
- **原用途**：加权双向特征金字塔（BiFPN），替代 PANet neck，提升多尺度特征融合质量
- **消融结果**：mAP50 = 0.9365（vs 无 BiFPN 的 0.9749），负贡献 **-3.84%**
- **归档原因**：391 张小数据集上额外参数导致过拟合，泛化性下降
- **归档时间**：2026-03

---

## `carafe.py`

- **来源**：CARAFE（ICCV 2019）
- **原用途**：内容感知特征重组（Content-Aware ReAssembly），替代 P2 分支的双线性上采样，理论上保留更多细节
- **消融结果**：集成进 SOTA 配置（`train_sota.py`）后训练不稳定，loss 震荡
- **归档原因**：与 RepVGG/BiFPN 等模块组合时梯度不稳定，已改回标准上采样
- **归档时间**：2026-03

---

## `repvgg.py`

- **来源**：RepVGG（CVPR 2021）
- **原用途**：结构重参数化——训练时多分支（3×3 + 1×1 + identity），推理时合并为单分支，提升速度
- **消融结果**：`train_sota` 系列（含 RepVGG backbone）全面低于基线
- **归档原因**：391 张小数据集上过拟合，结构重参数化在本任务无收益；推理加速的前提是大规模部署，本场景不适用
- **归档时间**：2026-03

## `legacy/`

### `legacy/attention.py`

包含两个早期注意力模块：

| 模块 | 来源 | 原用途 | 归档原因 | 归档时间 |
|------|------|--------|---------|---------|
| `EMA` | ICASSP 2023 | 多尺度通道空间注意力，项目早期方案 | 引入额外参数，导致早停（epoch 43），被零参数 SimAM 替代 | 2026-03 |
| `CA` | CVPR 2021 坐标注意力 | 嵌入精确位置信息的通道注意力，精度优于 EMA | 同样有额外参数，小数据集过拟合，被 SimAM 替代 | 2026-03 |

### `legacy/inner_iou.py`

- **来源**：Inner-IoU（2023）
- **原用途**：用辅助 bbox 计算 IoU，改善小目标稀疏梯度问题
- **归档原因**：被 SA-NWD 替代（Wasserstein 距离从根本上解决 box 不重叠时零梯度问题），未进入消融实验
- **归档时间**：2026-03
