## 中文版（精简）

1. 我们提出一种面向反无人机场景的尺度自适应 Wasserstein 定位度量（SA-NWD），并在损失、标签分配、NMS 全流程中一致化集成。
2. 我们基于标注噪声正则化动机设计自适应带宽 C_adapt，通过固定 C 对比（Exp D）和方向对比（Exp E）消融验证设计合理性，不依赖未经证实的理论最优性声明。
3. 我们构建了以 P2 小目标检测头为核心的轻量化架构，在提升极小目标召回的同时保持实时推理。
4. 通过系统消融（主链 + 自适应 C 方向验证 + 超参数敏感性）和跨数据集验证，证明方法的稳健性与泛化能力。

## English Version (Paper-ready)

1. We propose SA-NWD, a scale-adaptive Wasserstein-based localization metric for anti-UAV small object detection, integrated consistently across loss, label assignment, and NMS.
2. We design the adaptive bandwidth $C_\text{adapt}$ under an annotation-noise regularization motivation, and validate the design choice through controlled ablations comparing fixed $C$ (Exp~D) and the theoretically-motivated reverse direction (Exp~E), without overstating theoretical optimality.
3. We develop a P2-head-centric lightweight detector that substantially improves tiny-object recall while maintaining real-time inference.
4. Systematic ablations (main chain, adaptive-$C$ direction, $k$ sensitivity) and cross-dataset evaluation demonstrate consistent and reproducible improvements.

## 更保守版本（适合审稿偏严格时）

1. We introduce a practical scale-adaptive Wasserstein localization strategy for anti-UAV small object detection.
2. We present a design-aligned adaptive bandwidth with explicit experimental validation of the scale-adaptation direction.
3. We combine a P2 detection head with SA-NWD loss to improve small-object sensitivity under a lightweight constraint.
4. Controlled ablations and cross-dataset evaluations show reproducible improvements over strong YOLO baselines.
