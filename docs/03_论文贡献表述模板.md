# 论文贡献表述模板（可直接替换）

## 中文版（精简）

1. 我们提出一种面向无人机小目标检测的尺度自适应 Wasserstein 定位度量，并在训练与推理流程中进行一致化集成。  
2. 我们给出与实现严格对齐的理论与工程设计，明确尺度建模方式及可稳定收敛的单阶段训练策略。  
3. 我们构建了轻量化检测架构（P2 + SimAM + PConv），在提升小目标召回的同时保持实时性。  
4. 通过多随机种子消融与跨设置验证，证明方法在精度、稳定性与效率上的一致增益。

## English Version (Paper-ready)

1. We propose a scale-adaptive Wasserstein-based localization formulation for UAV small object detection and integrate it consistently across training and inference stages.  
2. We provide a theory-driven yet implementation-aligned design with explicit scale handling and stable single-stage optimization.  
3. We develop a lightweight detector variant (P2 + SimAM + PConv) that improves small-object recall while preserving real-time efficiency.  
4. Extensive multi-seed ablations and cross-setting evaluations demonstrate consistent gains in accuracy, robustness, and efficiency.

## 更保守版本（适合审稿偏严格时）

1. We introduce a practical scale-adaptive Wasserstein localization strategy for UAV small-object detection.  
2. We present a consistent method-to-implementation pipeline and validate stable one-stage training behavior.  
3. We combine P2, SimAM, and PConv to improve small-object sensitivity under lightweight constraints.  
4. Controlled ablations and fairness-aware comparisons show reproducible improvements over strong YOLO baselines.
