# Response to Reviewers (Template)

## Cover Note

Dear Editor and Reviewers,  
Thank you very much for your constructive comments. We have thoroughly revised the manuscript and conducted additional experiments to address all concerns. All modifications are reflected in the revised manuscript and supplementary materials. Point-by-point responses are provided below.

---

## Reviewer #1

### Comment 1
Novelty is unclear; the method appears to be a combination of existing modules.

### Response
Thank you for this important comment. We agree that the original manuscript did not clearly separate methodological novelty from engineering components. In the revision, we explicitly define our core novelty as the scale-adaptive Wasserstein formulation and its unified integration into assignment/regression/post-processing, while P2/SimAM/PConv are presented as complementary architectural enhancements.

### Changes in Manuscript
- Rewrote contribution list in Introduction.
- Reorganized Method section to distinguish:
  - core metric/loss contribution
  - architecture-level auxiliary modules
- Updated ablation table to isolate each component and the standalone gain of SA-NWD.

---

### Comment 2
The theoretical derivation is inconsistent with implementation (different forms of \(C(s)\)).

### Response
We appreciate this careful observation. We unified notation and retained one consistent scale definition and one final normalization form throughout the paper. We also added an explicit theorem-to-code mapping subsection.

### Changes in Manuscript
- Unified \(s\) definition and \(C(s)\) expression in Method.
- Removed conflicting equations and ambiguous wording.
- Added implementation note with exact training-time formula and hyperparameters.

---

### Comment 3
Experimental protocol is not sufficiently rigorous (single run, no variance).

### Response
We agree. We added 3-seed experiments for all key settings and now report mean ± std to reflect statistical stability.

### Changes in Manuscript
- Updated main comparison table and ablation table with mean ± std.
- Added per-seed metrics in supplementary material.
- Added discussion on variance and robustness.

---
