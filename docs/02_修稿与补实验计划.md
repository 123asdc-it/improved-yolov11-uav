## 立即可执行（本地，今天）

```bash
# 1. 转换 DUT Anti-UAV（5分钟）
python scripts/convert_dut_to_yolo.py
# → datasets/dut/  约10GB

# 2. 解压并转换 AI-TOD（~30分钟）
# 先解压到 datasets/aitod_raw/，然后：
python scripts/convert_aitod_to_yolo.py \
    --src datasets/aitod_raw \
    --dst datasets/aitod \
    --filter-xview
# → datasets/aitod/  约15GB（DOTA部分~18K张）

# 3. rsync 到服务器
rsync -avz datasets/dut/ root@server:/root/drone_detection/datasets/dut/
rsync -avz datasets/aitod/ root@server:/root/drone_detection/datasets/aitod/
rsync -avz scripts/ root@server:/root/drone_detection/scripts/

# 4. 服务器：Phase 1 完成后启动 DUT 队列
screen -S dut_queue
bash scripts/run_dut_queue.sh 2>&1 | tee logs/dut_queue.log
```
