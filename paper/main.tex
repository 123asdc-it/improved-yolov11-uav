\begin{abstract}
Unmanned Aerial Vehicle (UAV) based object detection has attracted significant attention due to its wide applications in surveillance, search and rescue, and traffic monitoring. However, detecting small objects from aerial perspectives remains challenging due to the limited pixel representation, complex backgrounds, and the demand for real-time inference on edge devices. In this paper, we propose an improved YOLOv11 framework specifically designed for UAV-based small object detection. Our method incorporates four key improvements: (1) a P2 small object detection head that leverages high-resolution feature maps at stride 4 to enhance small target detection; (2) an Efficient Multi-Scale Attention (EMA) module integrated into the backbone for better feature representation with minimal computational overhead; (3) Partial Convolution (PConv) based lightweight feature extraction blocks in the neck to reduce model parameters and improve inference speed; and (4) a BiFPN-style weighted feature fusion mechanism that enables adaptive multi-scale information aggregation. Ablation studies validate the effectiveness of each proposed component.
\end{abstract}

% ======================== I. Introduction ========================
\section{Introduction}

\IEEEPARstart{U}{nmanned} Aerial Vehicles (UAVs) have been increasingly deployed for various visual perception tasks, including surveillance, environmental monitoring, disaster assessment, and autonomous navigation \cite{zhu2021detection}. Object detection from UAV-captured imagery plays a critical role in these applications. However, compared to conventional ground-level object detection, UAV-based detection faces unique challenges:

\begin{itemize}
  \item \textbf{Small object scale}: Objects in aerial images often occupy only a few pixels due to the high altitude of UAVs, making them difficult to detect with standard detection heads designed for medium and large objects.
  \item \textbf{Complex backgrounds}: Aerial images contain cluttered backgrounds with varying textures, illumination conditions, and occlusions.
  \item \textbf{Real-time requirements}: UAV platforms typically have limited computational resources, demanding lightweight and efficient detection models.
\end{itemize}

Recent advances in deep learning-based object detection have led to remarkable progress, particularly with the YOLO (You Only Look Once) series \cite{redmon2016you}. The latest YOLOv11 \cite{yolo11_ultralytics} achieves state-of-the-art performance in general object detection with an efficient architecture. However, its default three-scale detection paradigm (P3/P4/P5) is not optimized for small objects, as the lowest detection scale at stride 8 still loses substantial spatial information for tiny targets.

To address these challenges, we propose an improved YOLOv11 framework that incorporates four complementary enhancements:

\begin{enumerate}
  \item \textbf{P2 Small Object Detection Head}: We extend the detection architecture with an additional P2 detection head operating at stride 4, which preserves fine-grained spatial features crucial for small object localization.
  \item \textbf{EMA Attention Module}: We integrate the Efficient Multi-Scale Attention (EMA) \cite{ouyang2023efficient} into the backbone network to enhance feature representation by capturing cross-channel and cross-spatial dependencies with minimal computational cost.
  \item \textbf{PConv Lightweight Design}: We replace standard convolution blocks in the neck with Partial Convolution (PConv) \cite{chen2023run} based modules to significantly reduce parameters and computational overhead.
  \item \textbf{BiFPN Weighted Fusion}: We adopt the Bi-directional Feature Pyramid Network (BiFPN) \cite{tan2020efficientdet} fusion strategy with learnable weights to improve multi-scale feature aggregation.
\end{enumerate}

The main contributions of this paper are summarized as follows:

\begin{itemize}
  \item We propose an improved YOLOv11 architecture tailored for UAV-based small object detection, achieving significant accuracy improvement while maintaining lightweight design.
  \item We design a four-head detection structure with P2 integration and lightweight PConv blocks, balancing detection accuracy and computational efficiency.
  \item We conduct comprehensive experiments and ablation studies on a UAV aerial dataset, demonstrating the superiority of the proposed method over existing approaches.
\end{itemize}

The rest of this paper is organized as follows: Section~\ref{sec:related} reviews related work. Section~\ref{sec:method} presents the proposed method in detail. Section~\ref{sec:experiments} describes the experimental setup and results. Section~\ref{sec:conclusion} concludes the paper.

% ======================== II. Related Work ========================
\section{Related Work}
\label{sec:related}

\subsection{YOLO Series for Object Detection}

The YOLO family of detectors has been widely adopted for real-time object detection since its introduction by Redmon et al. \cite{redmon2016you}. Subsequent versions, including YOLOv3 \cite{redmon2018yolov3}, YOLOv5 \cite{yolov5}, YOLOv7 \cite{wang2023yolov7}, and YOLOv8 \cite{yolov8_ultralytics}, have progressively improved detection accuracy and inference speed through architectural innovations such as CSP (Cross Stage Partial) connections, PANet feature fusion, and anchor-free detection heads.

YOLOv11 \cite{yolo11_ultralytics} introduces the C3k2 module and C2PSA (Cross Stage Partial with Spatial Attention) for enhanced feature extraction, achieving a favorable balance between accuracy and efficiency. However, its default three-scale detection architecture limits its performance on small objects commonly encountered in aerial imagery.

\subsection{Small Object Detection}

Small object detection remains a fundamental challenge in computer vision. Feature Pyramid Networks (FPN) \cite{lin2017feature} introduced multi-scale feature fusion to address scale variation, while PANet \cite{liu2018path} further enhanced information flow through bottom-up path augmentation. Several works have proposed adding higher-resolution detection heads \cite{yang2019scrdet, duan2021coarse} to improve small object detection. TPH-YOLOv5 \cite{zhu2021tph} added a prediction head at a higher resolution for drone-captured scenarios. These methods demonstrate that leveraging high-resolution features is critical for detecting small targets.

\subsection{Attention Mechanisms}

Attention mechanisms have shown remarkable effectiveness in enhancing feature representation. SE-Net \cite{hu2018squeeze} introduced channel attention, while CBAM \cite{woo2018cbam} combined channel and spatial attention. The Efficient Multi-Scale Attention (EMA) \cite{ouyang2023efficient} further advances attention design by employing parallel multi-scale sub-networks to capture cross-channel information with minimal computational overhead, making it particularly suitable for real-time applications.

\subsection{Lightweight Network Design}

Lightweight network architectures aim to reduce computational costs while preserving accuracy. MobileNet \cite{howard2017mobilenets} introduced depthwise separable convolutions, while GhostNet \cite{han2020ghostnet} generates more features from cheap operations. FasterNet \cite{chen2023run} proposed Partial Convolution (PConv), which applies convolution only to a fraction of input channels, achieving significant speedup with negligible accuracy loss.

\subsection{P2 Small Object Detection Head}

Standard YOLO detectors employ three detection heads at strides 8 (P3), 16 (P4), and 32 (P5), corresponding to small, medium, and large object scales. For UAV imagery where targets may occupy as few as $20 \times 20$ pixels in a $1920 \times 1080$ image, the P3 feature map at stride 8 (producing a $160 \times 135$ grid for an input of $1280 \times 1280$) may still provide insufficient resolution.

We introduce an additional P2 detection head operating at stride 4, which produces a feature map with $4\times$ the spatial resolution of P3. As illustrated in Fig.~\ref{fig:p2head}, the P2 branch is constructed by:

\begin{enumerate}
  \item Upsampling the P3 features and fusing with the backbone's P2-level features (layer 2 output, stride 4).
  \item Processing through a PConv-C3k2 block to generate the P2 detection features.
  \item Connecting to a dedicated Detect module alongside P3, P4, and P5.
\end{enumerate}

The four-head design significantly improves the recall of small objects while maintaining detection capability for larger targets through the P4 and P5 heads.

\subsection{Efficient Multi-Scale Attention (EMA)}

To enhance the backbone's feature representation capability without significantly increasing computational cost, we integrate the Efficient Multi-Scale Attention (EMA) \cite{ouyang2023efficient} module after key C3k2 blocks at the P3 and P4 levels.

The EMA module operates by first reshaping input features into groups, then capturing spatial dependencies through parallel horizontal and vertical pooling operations. Specifically, for an input feature $\mathbf{X} \in \mathbb{R}^{C \times H \times W}$, EMA:

\begin{enumerate}
  \item Divides the channels into $G$ groups: $\mathbf{X}_g \in \mathbb{R}^{(C/G) \times H \times W}$.
  \item Applies adaptive average pooling along height and width dimensions separately to generate spatial attention maps.
  \item Fuses the spatial attention through a $1 \times 1$ convolution and a $3 \times 3$ convolution path.
  \item Computes cross-attention between the two paths using softmax-normalized matrix multiplication.
  \item Applies the resulting attention weights to modulate the input features.
\end{enumerate}

The output can be formulated as:
\begin{equation}
  \mathbf{Y} = \mathbf{X}_g \odot \sigma(\mathbf{W}_{\text{cross}})
\end{equation}
where $\sigma$ is the sigmoid activation and $\mathbf{W}_{\text{cross}}$ represents the cross-attention weights from dual-path interaction.

By placing EMA at the P3 and P4 levels, the backbone gains enhanced ability to focus on informative regions while suppressing background noise, which is particularly beneficial for small object detection in cluttered aerial scenes.

\subsection{Partial Convolution (PConv) Lightweight Design}

To offset the computational overhead introduced by the additional P2 detection head, we incorporate Partial Convolution (PConv) \cite{chen2023run} in the neck's feature processing blocks.

Standard convolution applies the kernel to all input channels, resulting in computational complexity of $\mathcal{O}(k^2 \cdot C_{\text{in}} \cdot C_{\text{out}} \cdot H \cdot W)$. PConv reduces this by convolving only a fraction ($1/n_{\text{div}}$) of the input channels:

\begin{equation}
  \mathbf{Y}_{\text{conv}} = \text{Conv}(\mathbf{X}_{[:C/n_{\text{div}}]}), \quad
  \mathbf{Y} = \text{Concat}(\mathbf{Y}_{\text{conv}}, \mathbf{X}_{[C/n_{\text{div}}:]})
\end{equation}

where $n_{\text{div}} = 4$ by default, reducing the convolution FLOPs by approximately $75\%$. The untouched channels retain the original feature information, preserving representational capacity.

We design a PConv-C3k2 block that replaces the standard C3k2 module in the neck (Fig.~\ref{fig:pconv}), achieving:
\begin{itemize}
  \item Approximately $15\text{--}25\%$ reduction in model parameters.
  \item Improved inference speed due to reduced floating-point operations.
  \item Comparable detection accuracy to standard convolution blocks.
\end{itemize}

\subsection{BiFPN Weighted Feature Fusion}

Standard feature fusion in YOLO uses simple concatenation, treating all input features equally regardless of their importance. We adopt the BiFPN \cite{tan2020efficientdet} weighted fusion strategy, which assigns learnable weights to each input feature map:

\begin{equation}
  \mathbf{O} = \text{Concat}\left(\frac{w_1}{\sum_i w_i + \epsilon} \cdot \mathbf{F}_1, \quad \frac{w_2}{\sum_i w_i + \epsilon} \cdot \mathbf{F}_2\right)
\end{equation}

where $w_i \geq 0$ are learnable scalar weights enforced through ReLU activation, and $\epsilon = 10^{-4}$ ensures numerical stability. This fast normalized fusion allows the network to adaptively emphasize more informative feature scales during training, improving the quality of multi-scale feature aggregation without adding significant computational overhead.

\section{Conclusion}
\label{sec:conclusion}

In this paper, we proposed an improved YOLOv11 framework for UAV-based small object detection. By integrating a P2 small object detection head, EMA attention modules, PConv lightweight convolution blocks, and BiFPN weighted feature fusion, our method achieves significant improvements in detection accuracy while maintaining computational efficiency. Experimental results on a UAV aerial dataset demonstrate that the proposed method outperforms the baseline YOLOv11n by X.X\% in mAP@0.5 with only X.XM parameters, meeting the real-time detection requirements for UAV applications. Ablation studies validate the effectiveness and complementary nature of each proposed component. Future work will explore extending this approach to multi-class detection and deploying the model on embedded UAV platforms.
